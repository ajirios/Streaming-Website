

# Chill Online

A web service for chatting with friends and accessing multimedia streaming with Pepper via Portable Native Client(pNaCl) modules.

# Domain

http://chillonline.net


# Dependencies

1. Java Server Pages (JSP)
2. JavaScript
3. C++
4. Pepper Plugin API (PPAPI)
5. MySQL / MariaDB
6. AJAX
7. XML
